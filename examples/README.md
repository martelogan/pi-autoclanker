# examples

This repo ships three example tiers:

- `targets/parser-quickstart/`: the real runnable parser target with packaged
  source and benchmark files
- `minimal/`: the smallest useful kickoff shape for `pi-autoclanker`
- `parser-demo-expanded/`: a fuller worked session after the extension has
  already materialized local files, eval state, and a candidate pool

Use `targets/parser-quickstart/` when you want a concrete app and benchmark to
run immediately. Use `minimal/` next to see the real beginner wrapper surface.
Use `parser-demo-expanded/` when you want to inspect the richer resumable
bundle around that same packaged target.
