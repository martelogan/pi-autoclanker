# examples

This repo ships two example tiers:

- `minimal/`: the smallest useful kickoff shape for `pi-autoclanker`
- `parser-demo-expanded/`: a fuller worked session after the extension has
  already materialized local files, eval state, and a candidate pool

Use `minimal/` first if you want to understand the real beginner surface. Use
`parser-demo-expanded/` when you want to inspect the richer resumable bundle.
